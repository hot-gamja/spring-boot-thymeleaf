Layout Test Page

새로운 Layout, Header, Sidebar 구조가 정상적으로 작동하는지 테스트하는 페이지입니다.

테스트 항목

  • Header: 상단에 검은색 헤더가 표시되어야 합니다
  • Hamburger Menu: 왼쪽 상단 햄버거 아이콘을 클릭하면 사이드바가 나타나야 합니다
  • Sidebar: 사이드바에는 홈, 소개, 게시글, Photo Gallery 메뉴가 있어야 합니다
  • Top Navigation: 넓은 화면에서는 상단 우측에 홈, 게시글, 소개, 로그인 메뉴가 표시됩니다
  • Responsive: 1024px 이상에서는 사이드바가 항상 고정되어 있어야 합니다
  • Footer: 하단에 푸터가 표시되어야 합니다

반응형 테스트

브라우저 창 크기를 조절하면서 다음을 확인하세요:

  • Desktop (1024px 이상): 사이드바가 항상 왼쪽에 고정
  • Tablet/Mobile (1024px 미만): 사이드바가 숨겨지고 햄버거 메뉴로만 열림
  • Mobile (768px 미만): 상단 네비게이션이 숨겨지고 모든 메뉴는 사이드바로 접근
