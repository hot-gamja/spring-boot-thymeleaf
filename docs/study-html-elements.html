4. data-* 속성 — 태그에 임의 데이터 붙이기

HTML 구조
data-theme 박스 (현재: light)

💡 CSS 선택자: [data-theme="dark"] — 속성 값으로 스타일을 분기합니다. JS에서는 element.setAttribute('data-theme', 'dark') 로 설정합니다.

5. <link disabled> — CSS 파일 켜고 끄기

이 박스의 테마가 CSS 파일로 제어됩니다

현재: 라이트 CSS 활성화
실제 사용 HTML (b.md 참고)
<link id="hljsLight" href="github.min.css"> <!-- 활성화 --> <link id="hljsDark" href="github-dark.min.css" disabled> <!-- 비활성화 -->
💡 disabled 속성이 있으면 CSS 파일은 다운로드되어 있지만 적용되지 않습니다. JS에서 linkEl.disabled = true/false 로 즉시 전환할 수 있습니다.

6. MutationObserver — DOM 변화 실시간 감시

감시 대상 박스 (class: 없음)



Observer 로그:
💡 MutationObserver는 이벤트 리스너와 다르게, 누가 클래스를 바꾸든 상관없이 변화 자체를 감지합니다. 테마 버튼 클릭, JS 코드, 외부 라이브러리 — 어느 쪽이 바꿔도 반응합니다.

7. <pre> / <code> 구조

박스(pre) vs 글자(code) 역할 분리
public class Hello {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}

위 박스에서:

  • <pre> — 배경색, 테두리, 패딩 (박스 역할)
  • <code> — 글자 색, background: transparent (글자 역할)
💡 <html> 상단의 .dark 토글 버튼(섹션 1)을 눌러보면 이 코드블록도 같이 색상이 바뀝니다.
